# -FANET-Enabled-Multi-Drone-Systems
Time Series Line Graphs
Connectivity Degree over Time: Records changes in the number of network neighbors per drone.

Detection Events over Time: Plots the rate of target acquisition events per drone in 3000 iterations.

Tracking Error over Time: Measures variations in tracking accuracy throughout the active phase of the mission.

Speed over Time: Captures the movement pattern of the swarm.

Heat Maps
Speed Heat Map: A sampled heat map illustrating the intensity of speed values for all 10 drones over time.

Error Heat Map: A sampled heat map indicating tracking errors within the swarm.

Average/Total Bar Graphs
Average Speed: Average speed value for each drone.

Average Tracking Error: Average tracking error per drone.

Total Detection Events: Number of detections per drone.

Average Connectivity Degree: Average number of connections per drone throughout the entire experiment.
Experimental Setup Details
Active Phase: The first 850 iterations make up the highly dynamic phase of the experiment during which drones track targets.

Inactive Phase: Iterations 851-3000 correspond to a period of hovering with no movement and low tracking error.
